# 📝 Minimalist Notes App

A sleek, modern, and dark-themed Notes application built with Flutter. Designed with a clean minimalist aesthetic to provide a seamless, lightweight, and distraction-free note-taking experience.

---

## ✨ Features

- **Clean UI/UX:** Dark-mode enthusiast design with high-contrast elements for optimal readability.
- **Full CRUD Operations:** Smoothly Create, Read, Update, and Delete your personal thoughts and tasks.
- **Local Storage (Hive):** Super-fast, lightweight, and reliable local data persistence using Hive database.
- **Localization Support:** Built-in localization features for seamless language handling.
- **Dynamic Date Tracking:** Notes are saved with precise timestamps automatically.
- **Quick Action Tools:** Built-in options to share notes or delete them instantly from the main feed.
- **Empty State Guidance:** Intuitive onboarding messages when no notes are available.

---

## 🛠️ Tech Stack & Architecture

- **Framework:** Flutter (Dart)
- **Database:** Hive (Local NoSQL Database)
- **Architecture:** Feature-First Modular Structure (`models`, `views`, `constants`)
- **UI Design:** Minimalist Custom Dark Theme
