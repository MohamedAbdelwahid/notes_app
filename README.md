# ScholarChat 🚀

تطبيق محادثة (Chat App) تم بناؤه باستخدام إطار العمل **Flutter**.

## 📌 عن المشروع
هذا المشروع هو تطبيق محادثة يهدف إلى تسهيل التواصل، مع التركيز على تصميم واجهة مستخدم (UI) نظيفة وسهلة الاستخدام.

## 🛠️ ما تم إنجازه حتى الآن:
* **UI Design:** تصميم صفحات تسجيل الدخول (Login) وإنشاء الحساب (Register).
* **Custom Widgets:** بناء مكونات مخصصة لإعادة الاستخدام مثل الأزرار وحقول النص.
* **Form Validation:** إضافة منطق التحقق من البيانات المدخلة قبل الإرسال.

## 🚀 خطط مستقبلية:
* ربط التطبيق بـ **Firebase** لإدارة المستخدمين والمحادثات الفورية.
* إضافة خاصية رفع الصور داخل المحادثة.
